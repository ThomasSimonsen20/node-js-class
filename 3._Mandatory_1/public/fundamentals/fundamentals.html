Variabler

CONST

- Variables defined with const cannot be Redeclared.
- Variables defined with const cannot be Reassigned.
- Så const, bruger man hvis man ikke skal ændre en værdi på variablen.
- Const variabler skal havde en værdi når de bliver declared.
- Hvis variablen er et object eller et array, kan man dog:
- Change the elements of const array & Change the properties of const object

LET

- Variables defined with let cannot be Redeclared.
- Variables defined with let must be Declared before use.
- Så man kan ikke redeclare let, men du kan reassign, så man bruger altså let, hvis man vil ændre på variables værdi senere hen.
- En god arbejdsmetode er at starte med at lave en variable const, og senere hen ændre til let, hvis man finder ud af at man vil ændre variablen.
