express

- Express er et node web framework, som gør det let at arbedje med HTTP i node.
- Bruges til at update en del af ens JSON object.
- F.eks. hvis vi vil updatere en memes, kan vi finde en memes vi vil update med ID, og derefter ændre den meme uden at ændre ID.

app.get

- .get er en http get request
- Bruges til f.eks. at sende informationer til client, kan være html filer, json objekter osv..
- Kan sende JSON object baseret på ID, eller sende et helt array

app.patch

- .patch er en http patch request
- Bruges til at update en del af ens JSON object.
- F.eks. hvis vi vil updatere en memes, kan vi finde en memes vi vil update med ID, og derefter ændre den meme uden at ændre ID.

app.post

- .post er en http post request
- Bruges til at create JSON object(primært)
- Før automatisk givet et ID, så skal kun give det informationer der skal være i objektet.
